Columns / grid

The grid system of template follows the mobile-first approach and accomodates up to 10 grid columns. It uses units with predefined classes inside each grid, which define the column width.

Usage

To create columns, first create a row that will contain columns (the columns container .uk-grid ), then add your desired number of columns with appropriate sizes (uk-width-*).

Take a look at following basic guide to see how easy is to set-up columns:

We built an intentional redundancy into each set of unit classes, so that for instance the .uk-width-5-10 class will work just as well as .uk-width-1-2.

Resposive Width

Template provides a number of very useful responsive widths classes. Basicall they work just like the usual width classes, except they are prefixed, so that they only come to effect at certain breakpoints. This is great to adjust your layout and content for different device sizes.

Contact Form

Template includes a contact form which is located at contact.html and validated via JavaScript.

Javascript to validate form

Above Java file identifies/matches fields via their ID so make sure you add an id to every field, ie. input id="fullname" name="fullname" type="text"

PHP Form Processor

This contact form includes an intelligent PHP form processor (send-email.php) that automatically detects and sends all fields inside form tags, so if you add/edit/delete fields in the contact form you won't need to configure anything in this PHP file.

The only thing you should configure to make contact form work is your email address, which must be included at send-email.php.

JavaScript

All custom JavaScript settings can be found at "js/custom.js". this is the file where you can manually edit/customize scripts should you want to.

Besides this JavaScript file template imports following JavaScript files located in folder "js":

  • custom.js - All custom Javascript settings.
  • uikit.js - UIkit framework for most of the elements used in template (tabs, animations, toggle, responsive navigation, etc.).
  • jquery.min.js - Javascript v1.10.2 library
  • owl.carousel.min.js - Carousel/Slider jQuery extensions.
  • modernizr.custom.js - Javascript library that detects HTML5 and CSS3 features in the user's browser.
  • isotope.js - Magic layout jQuery plugin.
  • jquery.validate.js - Form validator used for contact page (contact form).
  • validation.js - Warning shown while validating contact form.
  • jquery.themepunch.plugins.min.js - Slider revolution plugin
  • jquery.themepunch.revolution.min.js - Slider revolution plugin
  • FlipClock.js - A fully extendible API to create any clock face you need.
