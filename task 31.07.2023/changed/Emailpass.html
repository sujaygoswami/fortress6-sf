Die Google Wallet App bietet die bestmöglichste Benutzererfahrung. Google Wallet Karten werden auf Servern gespeichert und über das Google-Konto der Nutzer*innen auf allen Geräten synchronisiert, die die Google Wallet App installiert haben.
Alternativ zur Google Wallet App können Sie auch die PassWallet-App nutzen (allerdings können wir hier die Schülerausweise nicht automatisch aktualisieren. Sie können eine Aktualisierung jedoch manuell auslösen.)

Sollten Sie den Pass mit den obigen Buttons nicht herunterladen können oder sollten die Buttons nicht richtig dargestellt werden, können Sie die Datei über diesen Link herunterladen:
Download Schülerausweis
2) Die Karte öffnen.

3) Nach dem Öffnen der Karte wird eine Vorschau der Karte angezeigt. Erst ein Klick auf “Zu Wallet hinzufügen” speichert die Karte in der Wallet-App.

4) Die Wallet-App am Smartphone prüfen.

Immer dann, wenn die Wallet-App geöffnet wird, kann der neue digitale Schülerausweis angezeigt werden. Die Rückseite des Schülerausweises kann geöffnet werden, indem auf die drei Punkte geklickt wird.

Manuelle Löschung Ihrer Daten:
Alternativ haben Sie die Möglichkeit, jederzeit Ihre Daten für den digitalen Schülerausweis löschen zu lassen. Beachten Sie bitte, dass Sie diesen Service dann zukünftig nicht mehr nutzen können. Schreiben Sie uns bitte mit Ihrer Kundennummer an datenschutz@energy-imaging.de
